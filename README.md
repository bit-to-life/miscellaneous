# 잡동사니 라이브러리

## 프로젝트 구성

| 프로젝트 이름 | 설명 |
|-------------|----|
| Miscellaneous | 기본 잡동사니. |
| Miscellaneous.Cryptography | 암호화 관련. AES 암호화/복호화, 해시유틸. |
| Miscellaneous.Database | EFCore 데이터베이스 관련. 페이징, 트랜젝션. |
| Miscellaneous.ExternalApi | 외부 API 클라이언트. |
| Miscellaneous.Image | SixLabors.ImageSharp를 사용하여 이미지 회전, 리사이즈, 포멧변환 등. |
| Miscellaneous.Spatial | 지오메트리 관련. 좌표계 변환, 거리 구하기, 지오메트리 생성 등. |
