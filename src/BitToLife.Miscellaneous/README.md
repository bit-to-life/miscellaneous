# BitToLife.Miscellaneous

기본적인 잡동사니 유틸리티 클래스들을 모아놓은 프로젝트입니다.

## Classes

### `CompressionUtil`

GZip 알고리즘을 이용하여 데이터를 압축하거나 압축된 데이터를 해제하는 클래스입니다.

### `DateTimeUtil`

날짜시간 관련 유틸리티 클래스입니다.

### `MathUtil`

수학 관련 유틸리티 클래스입니다.

### `PaginatedSet`

페이지 세트 클래스입니다.

### `PhoneNumberUtil`

전화번호 관련 유틸리티 클래스입니다.

### `RandomUtil`

랜던 유틸리티 클래스입니다.

### `SequentialNumberUtil`

연속되는 숫자를 생성하는 유틸리티 클래스입니다.
